Key Features:

   - Create, edit, delete bots

2. Channel Integration
   - Telegram: Use @BotFather to get token
   - Discord: Create app in Developer Portal
   - WhatsApp: Meta Business API required
   - Slack: Create Slack App with Bot Token
   - Web Widget: Embed with script tag

3. Knowledge Base (RAG)
   - Upload PDF, DOCX, TXT, MD files
   - Vector embeddings for semantic search
   - Auto-chunking and indexing

4. Autonomous Agents
   - Goal-based AI agents
   - Tool execution (web, email, API, file)
   - Memory and context retention

5. Voice AI
   - Twilio integration
   - Real-time transcription
   - Text-to-speech responses

6. Flow Builder
   - Visual drag-and-drop interface
   - Node types: Start, Message, AI, Condition, Action, End
   - Branch based on user input

API Endpoints:
- POST /api/bots - Create bot
- GET /api/bots - List bots
- GET /api/bots/:id - Get bot details
- PUT /api/bots/:id - Update bot
- DELETE /api/bots/:id - Delete bot
- POST /api/bots/:id/ai/chat - Send message to bot

Authentication:
- Register: POST /api/auth/register
- Login: POST /api/auth/login (returns JWT)
- Use header: Authorization: Bearer <token>

Rate Limits:
- Free: 20 req/min, 1,000/day
- Pro: 100 req/min, 10,000/day
- Enterprise: 500 req/min, 100,000/day

Common Issues:
1. "Invalid token" - Token expired, login again
2. "Rate limit exceeded" - Wait 1 minute
3. "Bot not found" - Check bot ID
4. "Channel not connected" - Verify credentials
5. "AI response failed" - Check API key in settings
